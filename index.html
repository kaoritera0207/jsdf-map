北部航空方面隊

担当：北海道〜北陸以北 ／ 司令部：三沢（青森）
基地名所在地主な部隊・装備役割と立地理由
三沢基地青森県三沢市北部航空方面隊司令部
F-35A 第3航空団 / 米空軍も駐留		空自唯一の日米共同使用航空基地。本州最北端に近く、ロシア機スクランブルの第一線。F-35Aで北方最前線の抑止力を担う
千歳基地北海道千歳市F-15J 第2航空団 / 特別航空輸送隊（政府専用機）北海道全域の防空拠点。新千歳空港に隣接し、ロシアとの距離が近い北の要衝
なぜ北海道・三沢か？ 北方はロシアとの地理的近接が最大の脅威要因。千歳はロシア機南下時の防衛ラインに位置し、三沢はさらに前に出た迎撃の「矢の先端」。冷戦期に対ソ防衛を最優先として大規模整備が進められた。

松島基地宮城県東松島市第4航空団（T-4練習機）/ ブルーインパルス操縦士養成拠点。ブルーインパルスの本拠地。2011年東日本大震災で被災したが復旧・再建
百里基地茨城県小美玉市F-2 第7航空団首都圏の防空を担う唯一の戦闘機基地。太平洋側からの侵入に対応。茨城空港と滑走路を共用
浜松基地静岡県浜松市航空教育集団司令部 / E-767（4機）/ PAC-3高射隊パイロット養成・術科教育の総本山。E-767で広域の航空脅威を探知し戦闘機を誘導する「空のコントロールタワー」
小牧基地愛知県小牧市KC-46A 空中給油・輸送機（4機）/ KC-767（4機）南西諸島有事での戦闘機の長距離展開・滞空時間延長を担う空中給油機の拠点
小松基地石川県小松市F-15J 第6航空団 / 飛行教導群（トップガン相当）日本海側唯一の戦闘機基地。日本海対岸（中国・ロシア）まで航空機で約1時間の最前線
なぜ小松か？ 日本海に面し、中国・ロシア両方に対応できる地理的中間点。他に日本海側の戦闘機基地がない中、北陸という位置が「西の三沢」として機能する。

海上自衛隊

2026年3月、護衛艦隊が「水上艦隊」に改編。約93隻・40万トンの艦艇が5大基地に分散配備。

5大基地（水上艦艇・潜水艦）

基地名所在地主な司令部・艦艇役割と立地理由
横須賀基地神奈川県横須賀市自衛艦隊司令部 / 護衛艦13隻・潜水艦11隻ほか海自最大・最重要の中枢基地。東京湾口を抑える要地。幕末から明治にかけて旧海軍が整備し、米海軍も駐留する日米共同の核
呉基地広島県呉市第2水上戦群 / 潜水艦約10隻旧海軍の大鎮守府を継承。瀬戸内海の出口として太平洋・東シナ海両方へのアクセスに優れる
佐世保基地長崎県佐世保市水陸両用戦機雷戦群司令部 / 第5航空群東シナ海に直接面した西方最重要基地。水陸両用作戦・機雷戦の司令部が2026年3月に新編。米海軍も隣接使用
舞鶴基地京都府舞鶴市第3水上戦群・護衛艦日本海に唯一面した海自基地。ロシア艦艇が通過する宗谷・津軽海峡を監視
大湊基地青森県むつ市大湊地区隊・護衛艦8隻津軽海峡・宗谷海峡のチョークポイントに隣接し、ロシア艦艇の常時監視を担う
5基地の地理的役割分担：
横須賀・呉 → 太平洋正面（潜水艦集中）
佐世保 → 東シナ海・大陸正面（西方の盾）
舞鶴・大湊 → 日本海・北方正面（ロシア艦艇の監視・阻止）

陸上自衛隊（5方面隊）

陸上総隊（市ヶ谷）が全国約160カ所の駐屯地・分屯地を一元指揮。

方面隊司令部

方面隊司令部担当エリア主な役割と立地理由
北部方面隊札幌駐屯地北海道全域陸自唯一の機甲師団（第7師団＝戦車部隊）を抱える。冷戦期に対ソ侵攻を想定して最重点整備
東北方面隊仙台駐屯地東北6県東日本大震災で統合任務部隊司令部を担った経験。日本海側からの上陸阻止ラインでもある
東部方面隊朝霞駐屯地（練馬区）関東・甲信越・静岡首都圏防衛が最大任務。人口の約40%を抱える地域の国家中枢保護と即応性が最優先
中部方面隊伊丹駐屯地東海〜中四国・近畿国土面積30%・2府19県を担当。3大都市圏を抱え有事の国民保護が重要任務
西部方面隊健軍駐屯地（熊本）九州・沖縄南西諸島・離島防衛のための水陸機動団（相浦駐屯地）を抱える。「南西シフト」の主役

注目駐屯地

駐屯地所在地役割
相浦駐屯地長崎県佐世保市水陸機動団（2,100名）の本拠。離島奪還・水陸両用作戦の専門部隊。米海兵隊に相当。佐世保の海自基地に隣接し海自輸送艦との連携による迅速な離島展開が可能
那覇駐屯地沖縄県那覇市第15旅団司令部。南西シフト強化の中心。宮古島・石垣島・与那国島にも部隊を展開
与那国駐屯地沖縄県与那国島台湾から約100km・中国大陸から約500kmの日本最西端。2016年に新設された沿岸監視隊（160名）が海上・空中の異常を常時監視する「哨戒の最前線」
防衛装備

防衛装備・兵器システム（2026年現在）

多層防衛：遠くで防ぎ、届かなければ近くで防ぐ重層的な構造。

L① 宇宙・早期警戒

装備概要
早期警戒衛星（DSP/SBIRS系）米国の衛星情報を共有。弾道ミサイルの発射を赤外線で即時探知。日本は自前の警戒衛星整備を進行中
E-767 早期警戒管制機浜松基地に4機配備。空中レーダーで広域の航空脅威を探知し戦闘機を誘導する「空のコントロールタワー」
E-2D アドバンスト・ホークアイ低空飛行するミサイルや飛行機も探知できる最新型艦載早期警戒機。三沢・那覇などに展開
宇宙作戦団（府中基地）スペースデブリや敵の衛星を監視。宇宙領域での作戦能力を整備中

L② ミッドコース迎撃（大気圏外）

装備搭載艦・配備先詳細
SM-3（スタンダードミサイル3）イージス護衛艦8隻弾道ミサイルを大気圏外で破壊する海上発射型迎撃ミサイル。こんごう型・あたご型各4隻に搭載済み
イージス・システム搭載艦（ASEV）建造中2027〜2028年就役（2隻）排水量約2万トン超のアジア最大級水上戦闘艦。SPY-7レーダーを搭載
GPI（滑空段階迎撃用誘導弾）日米共同開発2030年代配備目標極超音速滑空兵器（HGV）を滑空段階で迎撃する次世代システム

L③ PAC-3（ターミナル迎撃）

装備配備先詳細
PAC-3（ペトリオットMSE）全国24個高射隊（浜松・入間・春日・那覇・市ヶ谷・宮古島ほか）射程約35km。弾道ミサイルを大気圏内で迎撃する最後の盾。移動式なので有事には増援展開も可能
SM-3で撃ち漏らした場合にPAC-3で対処する二段構え。北朝鮮発射から沖縄まで約10分という短時間対処を迫られるため常時即応態勢を維持。

L④ 戦闘機（航空優勢）

機種保有数・計画主な配備基地役割
F-35A約100機調達計画三沢・千歳（順次）第5世代ステルス戦闘機。F-4の後継。高いステルス性と情報統合能力で単機制空・対地・電子戦に対応
F-35B 2025年〜42機調達計画新田原基地 / いずも型護衛艦短距離離陸・垂直着陸が可能。いずも型「空母化」と一体で推進。2025年8月に初の3機が新田原に到着
F-15J/DJ約200機（近代化改修中）千歳・小松・那覇・新田原現在の数量上の主力。ロシア・中国機へのスクランブルの主役
F-2約91機百里・三沢・築城日米共同開発の対艦攻撃特化機。2035年頃から退役開始見込み
GCAP（次期戦闘機）日英伊共同約100機 / 2035年目標開発中F-2の後継となる第6世代戦闘機。AIによる自律飛行・無人機編隊・極超音速対応を目指す
「いずも型護衛艦の空母化」：「いずも」「かが」の2隻が順次改修され、F-35Bを搭載・運用できる多用途護衛艦（CVM）に転換中。かがは2024年10月の試験でF-35Bの発着艦能力を確認済み。戦後日本で初の「艦上戦闘機運用能力」の獲得となる。
